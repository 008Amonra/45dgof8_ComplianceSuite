🐧 Linux Setup

Quick Start

curl -fsSL https://45dgof8.com/deploy-agent-ops.sh -o deploy.sh
chmod +x deploy.sh
./deploy.sh you@example.com

Step-by-Step

1. Install Docker

curl -fsSL https://get.docker.com | bash
sudo usermod -aG docker $USER
# Log out and back in for group to take effect

2. Start n8n

mkdir -p ~/agent-ops/n8n && cd ~/agent-ops/n8n
curl -fsSL https://45dgof8.com/docker-compose.yml -o docker-compose.yml
docker compose up -d
# Open http://localhost:5678

3. Create account

Open http://localhost:5678 → sign up with your email.

4. Install cloudflared tunnel

curl -fsSL https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64 -o cloudflared
chmod +x cloudflared
sudo mv cloudflared /usr/local/bin/
nohup cloudflared tunnel --url http://localhost:5678 &
# Find your tunnel URL: grep -oP 'https://[^.]+\.trycloudflare\.com' ~/agent-ops/tunnel.log

5. Install Python + FFmpeg

pip3 install --user numpy
sudo apt install ffmpeg

6. Start Command Center

mkdir -p ~/agent-ops/command-center
python3 ~/agent-ops/command-center/server.py &
# Open http://localhost:42042

7. Enable MCP (optional, for OpenCode)

In n8n: Settings → MCP Server → Enable. Create an API key, add to ~/.opencode/opencode.json.

🪟 Windows Setup

Quick Start

curl -fsSL -o deploy.bat https://45dgof8.com/deploy-agent-ops.bat
deploy.bat
# Or just double-click deploy.bat in Explorer

Step-by-Step

1. Install Docker Desktop

Download from docker.com — install, start, wait for the whale icon.

2. Start n8n

Open PowerShell and run:

mkdir %USERPROFILE%\agent-ops\n8n
cd %USERPROFILE%\agent-ops\n8n
curl -fsSL -o docker-compose.yml https://45dgof8.com/docker-compose.yml
docker compose up -d
# Open http://localhost:5678

3. Create account

Open http://localhost:5678 in your browser → sign up.

4. Install cloudflared tunnel

curl -fsSL -o cloudflared.exe https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-windows-amd64.exe
cloudflared.exe tunnel --url http://localhost:5678
# Tunnel URL appears in the window. Keep it open.

5. Install Python + FFmpeg

python -m pip install numpy
# FFmpeg: Download ffmpeg.exe from ffmpeg.org
# Add the folder with ffmpeg.exe to your PATH

6. Start Command Center

mkdir %USERPROFILE%\agent-ops\command-center
python %USERPROFILE%\agent-ops\command-center\server.py
# Open http://localhost:42042

7. Note on AI agents

OpenCode does not run on Windows. Use the n8n web UI directly (http://localhost:5678) to build and run workflows. Everything works the same — just click instead of chat.

What Each Piece Does

⚙️ n8n — The Workflow Engine

Connects to 400+ services (Telegram, email, OpenAI, Google). You build visual workflows — drag, drop, connect. Runs in Docker on all platforms.

🔌 MCP — The AI Bridge

Lets your AI agent (OpenCode) control n8n directly. Say "run the video workflow" and it executes. Linux/macOS only — use the n8n UI on Windows.

🌐 Cloudflare Tunnel — Web Access

Makes your local n8n accessible from anywhere via a *.trycloudflare.com URL. No port forwarding needed.

📊 Command Center — Dashboard

Zero-dependency Python dashboard showing service status. Auto-refreshes. Works on every OS.

🎵 Music Generators

Python scripts that synthesize 30s tracks from scratch — no samples, no GPU. numpy + FFmpeg required.
